#Automation Bot 🤖

บอทสำหรับกรอกข้อมูลอุณหภูมิอัตโนมัติบนเว็บไซต์สร้างขึ้นด้วย Python, Flask และ Playwright โดยมีหน้าจอ Web UI (Glassmorphism Design) ที่สวยงามและใช้งานง่าย สามารถรันผ่านคอมพิวเตอร์และสั่งงานผ่านเบราว์เซอร์ได้ทันที

## 🌟 ฟีเจอร์หลัก (Features)
- **Web UI:** หน้าควบคุมผ่านเบราว์เซอร์ ใช้งานง่ายบนมือถือและคอมพิวเตอร์
- **Multi-Field Support:** รองรับการกรอกทั้งช่องอุณหภูมิ (T) และช่องขับถ่าย/อาการ (U) พร้อมกันในคราวเดียว
- **Custom Value Picker:** Popup เลือกอุณหภูมิต่ำสุด-สูงสุดแบบแถบเลื่อน (Slider) รวมถึงสามารถเลือกค่า U ได้ (0-4, สุ่ม, หรือข้ามการกรอก)
- **Smart Navigation:** ระบบเปลี่ยนหน้าอัตโนมัติ รองรับการทำงานหลายหน้า
- **Real-time Logs:** แสดงสถานะการทำงานแบบสดๆ (Live Streaming) บนหน้าเว็บ พร้อมระบบเลื่อนข้อความอัตโนมัติ
- **Headless Toggle:** สวิตช์สำหรับเปิด/ปิด การแสดงหน้าต่างเบราว์เซอร์ขณะบอททำงาน
- **Smart Click:** ค้นหาช่อง U และ T ในตาราง พร้อมคลิกตัวเลขทศนิยมที่ซับซ้อนใน Popup ได้อย่างแม่นยำ

## 📋 ข้อกำหนดเบื้องต้น (Prerequisites)
- [Python 3.10](https://www.python.org/downloads/) หรือใหม่กว่า (แนะนำให้ติ๊กถูกที่ `Add Python to PATH` ตอนติดตั้งด้วย)
- ระบบปฏิบัติการ Windows (เนื่องจากมีไฟล์ `.bat` สำหรับเริ่มต้นระบบอย่างรวดเร็ว)

## 🛠️ วิธีการติดตั้งและใช้งาน (Installation & Usage)

1. **ติดตั้ง Dependencies**
   เปิด Command Prompt (หรือ Terminal) ในโฟลเดอร์นี้ แล้วรันคำสั่ง:
   ```bash
   pip install -r requirements.txt
   ```

2. **ดาวน์โหลดเบราว์เซอร์สำหรับ Playwright**
   รันคำสั่งนี้เพื่อดาวน์โหลดเบราว์เซอร์ Chromium:
   ```bash
   playwright install chromium
   ```

3. **เริ่มต้นระบบ (Run the App)**
   - ดับเบิลคลิกที่ไฟล์ `start.bat` 
   - ระบบจะเปิดหน้าต่างสีดำทิ้งไว้ (นี่คือเซิร์ฟเวอร์หลังบ้าน ห้ามปิด!)
   - เซิร์ฟเวอร์จะรันอยู่ที่: `http://127.0.0.1:5000`

4. **การใช้งานบอท**
   - เปิดเว็บเบราว์เซอร์ (Chrome, Edge ฯลฯ) แล้วพิมพ์ URL: `http://127.0.0.1:5000`
   - ใส่ชื่อผู้ใช้ (Username) และ รหัสผ่าน (Password)
   - กำหนดช่วงลำดับที่ต้องการกรอก (เช่น 1 ถึง 5)
   - เลือกช่วงเวลา และคลิกเลือกอุณหภูมิต่ำสุด-สูงสุด
   - กดปุ่ม **"เริ่มการทำงาน"** แล้วนั่งดูบอทจัดการให้ได้เลย!
Developed as an automated assistant to reduce repetitive manual data entry tasks.*
